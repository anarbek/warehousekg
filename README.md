# WarehouseKG

A warehouse management system built with a .NET backend, an Angular web client, and a Flutter mobile app, deployed via Docker.

## Tech Stack

- **Backend:** .NET
- **Web:** Angular
- **Mobile:** Flutter
- **Infra:** Docker
